不足：稍微慢

优点：快

js命名

声明变量：var

单一var：

var a = 10,

b = 20,

c = 30;

命名规则

  1. 变量名必须以英文字母“_(下划线)”,“$”开头。
  2. 变量名可以包含英文字母“_(下划线)”,“$”，数字。
  3. 不可以用系统的关键字、保留字作为变量名

基本语法：

值类型

原始值: Number,String,Boolean,undefined,null

Number: var  b  =  3

String: 字符串    var  b  =  "我是字符串"

Boolean：true/flase.

undefined:未给变量赋值

null：空值

引用值：array,object,function,...,date,...

语句基本规则

1.语句后面要用分号结束

2.语法错误会引发后续代码终止，但不会影响其他代码块行驶。

3.书写格式要规范，“=”，“+”，“-”，“/”两边都应有空格。

运算符

运算操作符:

"+"

1.数学运算，字符串相连

2.任何数据类型加字符串都等于字符串

优先级“=”最弱，“（）“优先级最强

a++等于a=a+1；

a+=10等于a=a+10

赋值的顺序，自右向左。运算顺序，自左向右

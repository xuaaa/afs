第十四天笔记

数组的定义：

  1. var arr=[];
  2. var arr=new Array(括号里若只填一个参数，则表示长度)

数组的读和写：

  1. arr[num]//不可以溢出读（结果是undefined）
  2. arr[num]=xxx;//可以溢出写

改变原数组：

  1. push:在数组里添加数据，在数组末尾添加。
  2. pop:将最后一位剪切。
  3. unshift：在数组前面加数据。
  4. shift：在数组前面减数据。
  5. reverse：将原数组逆转
  6. splice：arr.splice（从第几位开始，截取多少的长度，在切口处添加新的数据。）

  7. 改变原数组进行排序；（1.必须写两形参。2.看返回值）

    1.当返回值为负数时，那么前面的数放在前面

    2.当返回值为正数时，那么后面的数放前面。

    3.当返回值为0时，不动

    var.sotr(funnction(a,b){ //return a-b 升序 //return b-a 降序 });

不改变原数组：

  2. jion(): 返回一个字符串。该字符串是通过把 arrayObject 的每个元素转换为字符串，然后把这些字符串连接起来,arrayObject.join(separator
  3. slice(): 如果数组是空的arrayObject.slice(start,end)
  1. *toSource()*toSource() 方法表示对象的源代码。 该原始值由 Array 对象派生的所有对象继承。
  2. toSource() 方法通常由 JavaScript 在后台自动调用，并不显式地出现在代码中。object.toSource()
  3. toString(): arrayObject 的字符串表示。返回值与没有参数的 join() 方法返回的字符串相同arrayObject.toString()
  4. toLocaleString(): 把数组转换为本地字符串。arrayObject.toLocaleString()
    5. valueOf()： valueOf() 方法返回 Array 对象的原始值。
  5. 该原始值由 Array 对象派生的所有对象继承。
    6. arrayObject.valueOf()
